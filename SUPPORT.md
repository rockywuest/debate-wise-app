# Support

Thanks for using Debate Wise.

## Where to Ask for Help

- Bug reports: use the Bug issue template
- Feature ideas: use the Feature request template
- Security concerns: follow `SECURITY.md` (private reporting only)

## What to Include

- clear problem summary
- exact reproduction steps
- expected vs actual behavior
- environment details (OS, browser, app version)
- logs/screenshots when available

## Response Expectations

This is an open-source project with limited maintainer bandwidth. Response time can vary, but well-scoped reports with reproduction details get handled fastest.
